# Pizza-Sales-Analysis-Dashboard

Project Overview

The Pizza Sales Analysis Dashboard is a data analytics project built using SQL and Power BI to analyze sales performance and customer ordering behavior.
This project focuses on transforming raw sales data into meaningful business insights through data cleaning, SQL queries, and interactive visualizations.
The dashboard helps stakeholders understand revenue trends, peak order times, best-selling pizzas, and category performance to support data-driven decision-making.

Objectives

Analyze overall sales performance
Identify highest and lowest selling pizzas
Examine order distribution by day and week
Analyze time-of-day ordering trends
Evaluate category-wise order contribution
Provide actionable business insights
